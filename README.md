## ESLint

Optionally, if you would like to try this out, add next lint as a script in your package.json file:

```bash
"scripts": {
    "build": "next build",
    "dev": "next dev",
    "start": "next start",
    "lint": "next lint"
},
```
Then run pnpm lint in your terminal:

```bash
pnpm lint
```
